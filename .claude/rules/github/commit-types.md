# commit type 一覧

コミットメッセージおよび PR タイトルで使用する `type` の定義。

| type       | 用途                                       |
| ---------- | ------------------------------------------ |
| `feat`     | 新機能・新しいファイルの追加               |
| `fix`      | バグ修正・誤記修正                         |
| `docs`     | ドキュメント・Markdown の追加・更新        |
| `refactor` | 動作を変えないリファクタリング             |
| `chore`    | 設定ファイル・スクリプト・依存関係変更     |
| `add`      | 既存カテゴリに収まらない単純なファイル追加 |
| `remove`   | ファイル・コードの削除                     |
| `test`     | テストの追加・修正                         |
| `ci`       | CI/CD パイプラインの変更                   |
