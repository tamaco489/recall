# 手動修正対象ルール一覧

`markdownlint-cli2 --fix` では自動修正できず、手動対応が必要なルール。

| ルール | 内容                         | 修正方針                                                            |
| ------ | ---------------------------- | ------------------------------------------------------------------- |
| MD024  | 同一内容の見出し重複         | 見出し文言を変えるか、見出しを外して本文に組み込む                  |
| MD033  | インライン HTML              | Markdown 記法に書き換える                                           |
| MD036  | 見出しの代わりに太字を使用   | `**テキスト**` → `## テキスト` に変換する                           |
| MD040  | コードブロックに言語指定なし | バッククォートブロックに言語を追加する (例: `text`, `bash`, `json`) |
| MD041  | ファイル先頭が h1 でない     | 先頭に `# タイトル` を追加する、または MD041 を無効化する           |
| MD051  | リンクフラグメントの不整合   | 参照先の見出しと一致するよう修正する                                |
