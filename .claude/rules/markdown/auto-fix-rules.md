# 自動修正対象ルール一覧

`markdownlint-cli2 --fix` で自動修正される主なルール。

| ルール | 内容                           |
| ------ | ------------------------------ |
| MD009  | 行末の余分なスペース           |
| MD010  | ハードタブ → スペースに変換    |
| MD011  | リンク記法の誤り               |
| MD012  | 連続する空行                   |
| MD019  | ATX 見出し後のスペース         |
| MD021  | ATX 見出しの両端スペース       |
| MD022  | 見出し前後の空行               |
| MD023  | 見出しのインデント             |
| MD026  | 見出し末尾の句読点             |
| MD027  | ブロック引用後の余分なスペース |
| MD028  | ブロック引用内の空行           |
| MD030  | リスト後のスペース             |
| MD031  | コードブロック前後の空行       |
| MD032  | リスト前後の空行               |
| MD034  | 裸の URL                       |
| MD037  | 強調マークアップ内のスペース   |
| MD038  | インラインコード内のスペース   |
| MD039  | リンクテキスト内のスペース     |
| MD044  | 固有名詞の大文字小文字         |
| MD049  | 強調スタイルの統一             |
| MD050  | 太字スタイルの統一             |
| MD053  | 未使用リンク定義               |
